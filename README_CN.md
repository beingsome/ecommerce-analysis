# 电商用户行为路径与购买转化归因分析

**项目类型**：独立数据分析实战项目  
**数据量**：987,370 条用户行为日志，覆盖 9,739 名用户（2017.11.25 - 2017.12.04）  
**工具**：Python (Pandas, Matplotlib, Seaborn)

---

## 🔍 核心洞察

- **颠覆认知的发现**：最高购买转化率来自仅"点击"的用户组（**77.55%**），而非包含加购/收藏的用户组。
- **业务结论**：该平台用户多为“即搜即买”型，收藏/加购更多是犹豫和比价信号。
- **运营建议**：优化直接下单路径加入限时购买、抓住凌晨 0-2 点的高转化时段、区分“引流款”与“利润款”进行商品运营。
